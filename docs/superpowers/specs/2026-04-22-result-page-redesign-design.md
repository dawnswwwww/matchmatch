# 结果页重新设计 - 设计规格

**日期**: 2026-04-22
**状态**: 已批准

---

## 1. 概念与愿景

结果页采用"简约数字 + 全屏弹出层"结构。首页只展示最核心的匹配得分和一句话总结，让用户第一时间获得成就感和情感共鸣；详细答题对比通过弹出层呈现，支持滑动翻页和快速跳转，不产生视觉压迫。

---

## 2. 设计语言

沿用现有 CSS 变量系统，不引入新颜色或字体。

---

## 3. 结果页首页

### 布局结构

```
┌─────────────────────────────────┐
│         [综合得分大数字]         │  ← 80px, 黑色, %后缀
│           [匹配标签]             │  ← 28px, bold, 描述词
│         [一句话总结]             │  ← 14px, gray, 如"你们在X道题上想法一致"
│                                 │
│  [得分]    [默契题数]  [性格匹配] │  ← 三个小指标卡片, 横向排列
│                                 │
│       [ 查看答题详情 ]           │  ← 主 CTA, 全宽圆角按钮
│                                 │
│    [分享结果]    [ 再开一局 ]    │  ← 次要 CTA, 横向排列
└─────────────────────────────────┘
```

### 指标卡片说明

| 指标 | 内容 |
|------|------|
| 综合得分 | 0-100% 匹配度 |
| 默契题数 | "X / Y 题一致" |
| 性格匹配 | 百分比 + 简短描述 |

## 4. 答题详情弹出层

### 布局结构

```
┌─────────────────────────────────────┐
│  [×]          答题详情               │  ← 顶部栏, 关闭按钮
├─────────────────────────────────────┤
│  ○ ○ ● ○ ○ ○ ○ ○ ○ ○               │  ← 横向滚动题目索引条
│  [1]  [2]  [3]  [4]  [5]            │     圆点颜色: 绿色=默契, 灰色=分歧
├─────────────────────────────────────┤
│                                     │
│  第 3 题 / 共 10 题                 │  ← 当前题目指示
│                                     │
│  "你更愿意把时间花在？"              │  ← 题目文本
│                                     │
│  ┌─────────┐       ┌─────────┐    │
│  │ A: 社交  │  vs   │ B: 独处  │    │  ← 选项卡片
│  │  ✓ 默契  │       │         │    │     选中项下方显示状态
│  └─────────┘       └─────────┘    │
│                                     │
│     [← 上一题]    [下一题 →]        │  ← 左右滑动提示 + 翻页按钮
│                                     │
├─────────────────────────────────────┤
│  [全部]  [默契]  [有分歧]           │  ← 底部 Tab 切换
└─────────────────────────────────────┘
```

### 题目卡片状态

- **匹配**: 选项卡片边框变绿色, 显示 "✓ 默契" 徽章
- **分歧**: 选项卡片边框保持默认, 显示 "✗ 分歧"

### 快速索引条交互

- 可横向滚动
- 点击任意圆点跳转至对应题目
- 当前题目圆点放大 + 高亮

### Tab 筛选逻辑

- **全部**: 显示所有题目, 按题号顺序
- **默契**: 只显示匹配= true 的题目
- **有分歧**: 只显示匹配= false 的题目

切换 Tab 后, 索引条同步更新高亮状态.

---

## 5. 组件清单

| 组件 | 描述 |
|------|------|
| `ResultSummary` | 首页结果总结区域, 包含得分 + 指标卡 + CTA |
| `IndicatorCard` | 单个小指标卡片 (图标 + 数值 + 标签) |
| `ResultDetailModal` | 全屏弹出层容器 |
| `QuestionIndexBar` | 横向滚动题目索引条 |
| `QuestionCard` | 弹出层内单题卡片 (题目 + 选项 + 状态) |
| `QuestionTabBar` | 底部 Tab 切换栏 |

---

## 6. 技术注意事项

- 弹出层使用原生 `dialog` 元素或全屏 `div` + `fixed` 定位
- 滑动翻页使用 `scroll-snap` 实现, 不引入额外动画库
- 题目数据从 `useGameStore` 获取, 无需新增 API
